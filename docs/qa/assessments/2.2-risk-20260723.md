## Risk Matrix

| Risk ID  | Description                                          | Probability | Impact     | Score | Priority |
| -------- | --------------------------------------------------- | ----------- | ---------- | ----- | -------- |
| TECH-001 | Not-found/invalid-id raises instead of structured result | Medium (2)  | Medium (2) | 4     | Medium   |
| TECH-002 | Unbounded `list_tickets` floods agent context       | Low (1)     | Medium (2) | 2     | Low      |
| DATA-001 | Non-deterministic list order surprises graders       | Low (1)     | Medium (2) | 2     | Low      |

## Mitigations

- **TECH-001:** Tools validate id and return structured `invalid_request`/`not_found`; never raise. Negative tests for both.
- **TECH-002:** `list_tickets` clamps `limit` to a max; returns `total`/`offset`/`limit`.
- **DATA-001:** List order inherits `Dataset` sort (scenario_id); deterministic assertion in tests.
