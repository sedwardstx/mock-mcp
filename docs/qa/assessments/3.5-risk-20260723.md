## Mitigations
- DATA-001: `check_dataset` telemetry-evidence check runs over ALL scenarios; fails if any category lacks matching evidence. Negative test proves it catches a gap.
- DATA-002: multi-round scenarios put benign rows in the "obvious" table and real evidence elsewhere; e2e test asserts the first query is inconclusive and the second reveals the cause.
- TECH-001: generator only owns 4..60; scenarios 1-3 are hand-authored and untouched.
- DATA-003: category-varied templates + distinct resource names; acceptable for teaching MVP; Epic 4 curates further.
