- **DATA-001:** Parse ISO-8601 with tz; test boundary cases (row inside/outside range).
- **TECH-002:** All row lists default empty → `telemetry: {}` still valid; regression covered by existing consistency suite.
- **DATA-002:** Schemas mirror architecture columns; acceptable for MVP.
