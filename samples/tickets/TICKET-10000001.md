# TICKET-10000001 — VM will not start (allocation failure)

- **Product:** Azure Virtual Machines
- **Persona:** windows_admin
- **Severity:** Sev2
- **Status:** Active
- **Resource:** `/subscriptions/00000000-0000-0000-0000-000000000001/resourceGroups/rg-prod/providers/Microsoft.Compute/virtualMachines/prod-web-01`

failure when attempting to start the virtual machine.
