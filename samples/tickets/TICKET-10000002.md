# TICKET-10000002 — App cannot reach backend (connections timing out)

- **Product:** Azure Networking
- **Persona:** azure_developer
- **Severity:** Sev2
- **Status:** Active
- **Resource:** `/subscriptions/00000000-0000-0000-0000-000000000002/resourceGroups/rg-api/providers/Microsoft.Compute/virtualMachineScaleSets/api-vmss` (instances `api-vmss_0`, `api-vmss_1`)

## Symptom

Developer reports their application on VMSS `api-vmss` can no longer reach the
backend database on port 1433; connections time out intermittently after a recent
network change.

## Latest signal

Outbound TCP 1433 flows show `action=Deny` matching a new NSG rule
`DenyDbOutbound` (see `samples/logs/nsg-deny-1433.log`).

> Cross-check with the live server: `get_ticket TICKET-10000002` →
> `get_ticket_resources` → `query_network_logs` (filter `action='Deny'`).
